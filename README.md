O aplicativo **Álbum da Copa** foi desenvolvido para fins educacionais e permite o gerenciamento de figurinhas digitais da Copa do Mundo.

Os usuários podem:

* Criar uma conta
* Realizar login
* Visualizar jogadores
* Marcar figurinhas como coletadas
* Pesquisar e filtrar figurinhas
